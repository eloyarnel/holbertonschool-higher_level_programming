Project Description

This project introduces the fundamental concepts of Python programming.
It focuses on learning how to interact with the Python interpreter, display output, work with strings, and follow the official Python coding style.

The project serves as a first step toward understanding Python syntax and best practices.Topics Covered
 Using the Python Interpreter

Running Python in interactive mode

Executing Python scripts from the command line

 Printing Text and Variables

Using the print() function

Printing strings and variable values

 Strings in Python

Creating string literals

String concatenation and formatting

 Indexing and Slicing

Accessing individual characters using indexes

Extracting substrings using slicing notation

 Python Coding Style (PEP 8)

Understanding the official Python style guide

Writing readable and consistent code

Checking code style using pycodestyle
